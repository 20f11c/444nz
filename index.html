八叉书库广告屏蔽

针对八叉书库页面进行广告清理和交互优化，移除干扰元素并提升阅读体验。

去广告 弹窗拦截 0 金币自动订阅
一键安装脚本

主要功能

  • 屏蔽页面广告 iframe / script / ins 及站内广告脚本
  • 清理推荐广告模块、广告图片与落地链接
  • 拦截广告域名相关弹窗（window.open）
  • 免费章节（0 金币）自动订阅，内置防抖保护

安装与更新

  1. 先安装浏览器扩展 Tampermonkey。
  2. 点击“一键安装脚本”完成安装。
  3. 脚本通过 @updateURL / @downloadURL 自动更新。

发布新版本时请递增 @version，如遇站点改版可按需更新匹配规则。
